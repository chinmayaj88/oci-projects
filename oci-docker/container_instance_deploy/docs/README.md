# OCI Container Instance: End-to-End Guide

This documentation walks you through building and deploying a small Flask app to an Oracle Cloud Infrastructure (OCI) Container Instance. It covers prerequisites, IAM permissions, networking (VCN), Oracle Cloud Infrastructure Registry (OCIR) setup, Docker build/push, deployment, testing, troubleshooting, and cleanup.

Contents:

1. [Prerequisites](./01-prereqs.md)
2. [IAM and Networking](./02-iam-and-networking.md)
3. [OCIR Registry and Docker Image](./03-registry-and-image.md)
4. [Deploy Container Instance](./04-deploy-container-instance.md)
5. [Validate and Troubleshoot](./05-validate-and-troubleshoot.md)
6. [Cleanup](./06-cleanup.md)

If you prefer a single quick path, read each doc in order. Most commands are given for Windows PowerShell. Replace placeholders like `<tenancy_ocid>` or `<region>` with your own values.
